C.3.2 A demonstration regarding setting up the process noise in the system and measurement noise 

C.3.2.1 Generate Gaussian random number
// Generate Gaussian random numbers
float U1 = (float)random(0, 32767) / 32767;
float U2 = (float)random(0, 32767) / 32767;
float gaussian_random = sqrt(-2 * log(U1)) * cos(2 * PI * U2);

// Scale by the square root of the variance
float noise(process) = gaussian_random * sqrt(Q);

C.3.2.2 Generate the measurement noise
// Add measurement noise
float noise(measurement) = gaussian_random * sqrt(R);
actual measurement z is generated by adding Gaussian noise, scaled by the measurement noise R, to the low-cost sensor reading.
